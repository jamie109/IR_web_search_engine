### 食用指南
#### 1. 运行spider.py，爬取网页内容。
#### 2. 运行content_inverted_index.py，对网页文本内容构造倒排索引。
#### 3. 运行page_rank.py，进行链接分析。
#### 4. 运行main.py，进行查询。

* 爬取网页，索引，链接分析
* 站内查询
* 通配查询
* 登录系统（用户信息保存、查询）
* 日志记录
* tkinter 前端
* 个性化查询（基于年龄）
* 个性化推荐（基于年龄）
